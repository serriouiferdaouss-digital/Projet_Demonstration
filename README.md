# Backend FastAPI avec GraphQL et Postgres

Ce projet présente une application backend FastAPI servant des API GraphQL à l'aide de Strawberry, connectée à une base de données PostgreSQL. Il inclut une authentification sécurisée utilisant JWT et le hachage de mot de passe avec Bcrypt.

## Structure du Projet

- `backend/` : Contient le code de l'application FastAPI, les modèles de base de données et la configuration Docker.
- `frontend/` : (Actuellement vide) Emplacement réservé pour l'application frontend.
